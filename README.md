# Ultrix-Shoe-Palace-E-Commerce-Platform
Full-stack e-commerce platform with Spring Boot backend and HTML, CSS, JS frontend. Browse products, manage a cart, and place orders. Designed to be simple, functional, and easy to extend for learning or development purposes.

## Overview
This project is designed as a learning and demonstration platform for building a complete online store. It focuses on clean architecture, simplicity, and functionality while providing an extendable foundation for more advanced e-commerce features.

## Features
- Browse products with detailed information  
- Add, remove, and update items in the shopping cart  
- Place orders and simulate checkout  
- Backend powered by Spring Boot for data management  
- Frontend built with HTML, CSS, and JavaScript for a user-friendly interface  

## Tech Stack
- **Backend:** Spring Boot, Java  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL
